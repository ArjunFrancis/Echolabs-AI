## Lead Capture & Qualification

### **Lead Capture Form (Recommended Fields):**
- Full Name
- Email Address (required)
- Company Name
- Job Title / Role
- Industry / Sector
- Company Size (employees)
- Specific Use Case / Interest
- Timeline for AI Implementation
- Current AI Maturity (exploring, piloting, deploying, optimizing)

### **Lead Scoring:**
**High-Value Leads (Priority 1):**
- UAE-based enterprise (500+ employees)
- C-level or VP-level title
- Financial services, government, healthcare sectors
- Timeline: Next 3-6 months
- Current maturity: Piloting or deploying

**Medium-Value Leads (Priority 2):**
- UAE-based mid-market (100-500 employees)
- Director or Manager title
- Telecom, energy sectors
- Timeline: 6-12 months
- Current maturity: Exploring or piloting

**Low-Value Leads (Nurture):**
- Small companies (< 100 employees)
- Individual contributors
- Timeline: 12+ months
- Current maturity: Exploring

## Usage Guidelines

### **For Marketing Team:**
1. Create dedicated landing pages for each asset
2. Implement lead capture forms with progressive profiling
3. Set up automated email nurture sequences
4. Track conversion metrics (landing page → download → consultation)
5. A/B test headlines, CTAs, form fields

### **For Sales Team:**
1. Prioritize high-value leads within 24 hours
2. Reference downloaded asset in outreach ("I see you downloaded our [Asset]...")
3. Offer value-add consultation (not sales pitch)
4. Use asset as conversation starter
5. Qualify using BANT framework (Budget, Authority, Need, Timeline)

### **For Product Team:**
1. Monitor asset usage data for product insights
2. Identify common use cases from ROI calculator inputs
3. Validate compliance pain points from checklist downloads
4. Prioritize prompt library sectors by download volume
